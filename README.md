# Multi-Threaded Chat Application

## Description
A Java-based multi-threaded chat application that enables real-time communication between multiple clients using socket programming and multithreading.

## Technologies Used
- Java
- Socket Programming
- Multithreading
- Client-Server Architecture

## Features
- Real-time messaging
- Multiple client support
- Concurrent client handling
- Client-server communication

  ## Requirements

- Java JDK 8 or above
- Visual Studio Code (VS Code)
- Java Extension Pack for VS Code
- Windows 10/11
- Internet or Local Network Connection
